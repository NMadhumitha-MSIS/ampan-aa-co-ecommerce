# AMPAN AURA & CO

## Project Overview
AMPAN AURA & CO is a comprehensive e-commerce platform dedicated to makeup and skincare enthusiasts. Our website offers a curated selection of high-quality beauty products, from everyday essentials to luxury items. Designed with the modern beauty consumer in mind, AMPAN AURA & CO combines sleek aesthetics with user-friendly functionality to create an immersive shopping experience.

## Project Structure
- `FINALLOGIN.html`: HTML file for login and registration forms
- `loginstyle.css`: CSS file for styling the login and registration pages
- `homepage.html`: HTML file for the main content of the website, including navigation, carousel, and product displays
